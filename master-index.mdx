## API Reference Index

### EVM APIs

#### Balances API
- **Endpoint**: `GET /v1/evm/balances/{address}`
- **Purpose**: Get token balances for EVM addresses
- **Documentation**: [Complete Guide](/api-reference#balances-api)
- **Examples**: [Usage Examples](/usage-examples#building-a-portfolio-tracker)
- **SDK Support**: [JavaScript](/sdk-documentation#balances) | [Python](/sdk-documentation#evm-methods) | [Go](/sdk-documentation#evm-methods-1)
- **Related Functions**: `getEvmBalances()`, `getAllBalances()`, `formatTokenAmount()`
- **Tags**: `#evm` `#balances` `#tokens` `#portfolio`

#### Activity API
- **Endpoint**: `GET /v1/evm/activity/{address}`
- **Purpose**: Get real-time activity feed for addresses
- **Documentation**: [Complete Guide](/api-reference#activity-api)
- **Examples**: [Activity Feed Implementation](/usage-examples#activity-feed-implementation)
- **SDK Support**: [JavaScript](/sdk-documentation#activity) | [Python](/sdk-documentation#evm-methods) | [Go](/sdk-documentation#evm-methods-1)
- **Related Functions**: `getEvmActivity()`, `pollForNewActivity()`, `formatActivityType()`
- **Tags**: `#evm` `#activity` `#transactions` `#realtime`

#### Transactions API
- **Endpoint**: `GET /v1/evm/transactions/{address}`
- **Purpose**: Get detailed transaction information
- **Documentation**: [Complete Guide](/api-reference#transactions-api)
- **Examples**: [Transaction Processing](/usage-examples#transaction-processing)
- **SDK Support**: [JavaScript](/sdk-documentation#transactions) | [Python](/sdk-documentation#evm-methods) | [Go](/sdk-documentation#evm-methods-1)
- **Related Functions**: `getEvmTransactions()`, `processTransactions()`
- **Tags**: `#evm` `#transactions` `#gas` `#details`

#### Collectibles API
- **Endpoint**: `GET /v1/evm/collectibles/{address}`
- **Purpose**: Get NFT and collectible balances
- **Documentation**: [Complete Guide](/api-reference#collectibles-api)
- **Examples**: [NFT Gallery Implementation](/usage-examples#nft-gallery-implementation)
- **SDK Support**: [JavaScript](/sdk-documentation#collectibles-nfts) | [Python](/sdk-documentation#evm-methods) | [Go](/sdk-documentation#evm-methods-1)
- **Related Functions**: `getEvmCollectibles()`, `formatNFTMetadata()`
- **Tags**: `#evm` `#nft` `#collectibles` `#metadata`

#### Token Info API
- **Endpoint**: `GET /v1/evm/token-info/{token_address}`
- **Purpose**: Get detailed token information
- **Documentation**: [Complete Guide](/api-reference#token-info-api)
- **Examples**: [Token Information Lookup](/usage-examples#token-information-lookup)
- **SDK Support**: [JavaScript](/sdk-documentation#token-information) | [Python](/sdk-documentation#evm-methods) | [Go](/sdk-documentation#evm-methods-1)
- **Related Functions**: `getEvmTokenInfo()`, `batchTokenLookup()`
- **Tags**: `#evm` `#tokens` `#metadata` `#info`

#### Token Holders API
- **Endpoint**: `GET /v1/evm/token-holders/{token_address}`
- **Purpose**: Get holders of specific tokens
- **Documentation**: [Complete Guide](/api-reference#token-holders-api)
- **SDK Support**: [JavaScript](/sdk-documentation#token-information) | [Python](/sdk-documentation#evm-methods) | [Go](/sdk-documentation#evm-methods-1)
- **Related Functions**: `getEvmTokenHolders()`
- **Tags**: `#evm` `#tokens` `#holders` `#distribution`

#### Supported Chains API
- **Endpoint**: `GET /v1/evm/supported-chains`
- **Purpose**: Get list of supported EVM chains
- **Documentation**: [Supported Chains Guide](/evm/supported-chains)
- **Related Functions**: `getSupportedChains()`, `getChainInfo()`
- **Tags**: `#evm` `#chains` `#networks` `#support`
